## Color Scheme (Dark Theme)

```
Background:     #1a1a2e (deep navy)
Surface:        #16213e (card backgrounds)
Primary:        #e94560 (accent, CTAs)
Secondary:      #0f3460 (secondary elements)
Text Primary:   #ffffff
Text Secondary: #a0a0a0
Success:        #4ade80 (natural 20)
Danger:         #f87171 (natural 1)
Dropped:        #6b7280 (grayed out)
```

## Interaction States

### Button States
```
Normal:    bg: #0f3460, text: white
Hover:     bg: #e94560, transform: translateY(-2px), shadow
Active:    bg: #c13a50, transform: translateY(0)
Disabled:  bg: #374151, text: #6b7280
```

### Input States
```
Normal:    border: 1px solid #374151
Focus:     border: 2px solid #e94560, glow
Error:     border: 2px solid #f87171
```

### Dice Result States
```
Normal:    bg: #16213e, border: 2px solid #0f3460
Kept:      bg: #16213e, border: 2px solid #e94560
Dropped:   bg: #1f2937, text: #6b7280, text-decoration: line-through
Nat 20:    border: 2px solid #4ade80, glow green
Nat 1:     border: 2px solid #f87171, glow red
```

## Animation Timing

```
Dice Tumble:     1.5s ease-out
Result Reveal:   0.3s ease-out
Button Press:    0.1s ease-out
Panel Expand:    0.2s ease-in-out
Modal Open:      0.2s ease-out
Notification:    0.3s ease-out
```
